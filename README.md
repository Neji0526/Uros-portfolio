# 👨‍💻 Personal Portfolio

My Personal Portfolio is a modern responsive web-based portfolio website designed to showcase my technical skills, projects, education, internship experience, and professional profile.

It provides a clean and interactive interface for recruiters and visitors to explore my work, download my resume, and contact me directly.

---

🔗 **Live Demo   :** [https://madhan-03.netlify.app/]  
🔗 **GitHub Repo :** [https://github.com/Madhan-03/portfolio.git]

---

# 🚀 Features

- 📱 Fully Responsive Modern Design.  
- ⌨️ Animated Typing Text Effect.  
- 👨‍💻 Professional About Me Section.  
- 🛠️ Technical Skills Showcase.  
- 💼 Featured Projects Section.  
- 🎓 Education & Internship Timeline.  
- 📄 Download CV Button.  
- 📬 Contact Form with EmailJS Integration.  
- 🌐 Social Media Integration.  
- 📊 Total Visitors Counter.  
- 🟢 Online / Offline Status Indicator.  
- 🕒 Live Time and Date Display.  

---

# 🛠️ Tech Stack

- **Frontend :** HTML, CSS, JavaScript  
- **Email Service :** EmailJS  
- **Deployment :** Netlify / GitHub
- **Resume Hosting :** GitHub / Google Drive  

---

# 🎯 Objective

The goal of this project is to build a professional digital portfolio that effectively presents my software development skills, academic achievements, and technical projects to recruiters and collaborators.
